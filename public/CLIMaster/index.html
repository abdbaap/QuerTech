Stunning Windows Terminal Customization on Windows 11

Elevate your command-line experience with a personalized, high-performance setup.

Elevate Your Terminal Experience

This tutorial serves as a comprehensive guide to customizing the Windows Terminal on Windows 11. Our primary goal is to transform the default, often bland, terminal into a visually stunning and highly functional command-line interface. By enhancing its visual appeal and integrating essential system information, we aim to significantly elevate your daily workflow.

The speaker in the analyzed video maintains an enthusiastic, informal, and highly practical tone, using relatable analogies and crystal-clear instructions to make this technical process accessible and engaging for everyone.

Executive Summary: Beyond the Default

This guide provides a step-by-step process for transforming your default Windows Terminal into a truly personalized and informative CLI dashboard. We'll achieve this by integrating a custom theme, a specialized Nerd Font, and the powerful Fastfetch system information tool. Prepare to move beyond standard aesthetics and embrace a command-line experience tailored to you.

Your Step-by-Step Customization Guide

1

Prerequisite: Install JetBrainsMono Nerd Font

Crucial Step: Without a Nerd Font, special characters and icons used in the customized terminal will not display correctly, leading to a suboptimal visual experience. The recommended font is JetBrainsMono Nerd Font.

You can download it from a reliable source like JetBrains Mono official page or Nerd Fonts website.

2

Terminal Theme Customization (Catppuccin Mocha)

  1. Access the speaker's GitHub repository for configuration files (e.g., `github.com/YourSpeaker/WindowsTerminalConfig`).
  2. Navigate to the /Terminal/settings.json file.
  3. Copy the raw JSON content provided in the repository.
  4. Open Windows Terminal settings, then select "Open JSON file" to directly edit its configuration.
  5. Replace the entire existing content of your local settings.json file with the copied code and save. This immediately applies the Catppuccin Mocha theme and the JetBrainsMono Nerd Font.
3

PowerShell Profile Management

Your PowerShell profile file (profile.ps1) is key for custom startup commands.

  1. Determine its location by executing $PROFILE in the Windows Terminal.
  2. If the profile file does not exist, create it using the command: 
    New-Item -Path $profile.CurrentUserAllHosts -Type File -Force
4

Fastfetch Installation and Configuration

  1. Install fastfetch using Winget: 
    winget install fastfetch
  2. Return to the speaker's GitHub repository and navigate to the /Fastfetch/ directory.
  3. Download the config.jsonc and ascii.txt files.
  4. Create a hidden directory named .config within your user profile directory (e.g., C:\Users\<Your_Username>\).
    • To make the folder hidden: Right-click .config, "Show more options," "Properties," check "Hidden," Apply, OK.
    • To view hidden files: In File Explorer, "View," "Show," enable "Hidden items."
  5. Inside .config, create another folder named fastfetch.
  6. Move the downloaded config.jsonc and ascii.txt files into C:\Users\<Your_Username>\.config\fastfetch.
  7. Edit config.jsonc: Locate the source path for the logo and replace %USERPROFILE% with your actual Windows username. Save.
5

Integrate Fastfetch with PowerShell Profile

  1. Go back to the speaker's GitHub repository: /PowerShell/Microsoft.PowerShell_profile.ps1.
  2. Copy the raw content of this PowerShell script.
  3. Open your local profile.ps1 file (from step 3), paste the copied code into it.
  4. Replace the %USERPROFILE% placeholder in the fastfetch -c ... line with your actual username.
  5. Save and close the profile.ps1 file.
6

Verification

Congratulations! Open a new tab in your Windows Terminal. You should now see:

  • The customized Catppuccin Mocha theme.
  • The crisp JetBrainsMono Nerd Font.
  • The Fastfetch output, complete with your custom ASCII art and essential system statistics.

Your terminal is now a personalized, powerful, and aesthetically pleasing command center!

Actionable Insights: The "So What"

Implementing these steps transforms a utilitarian CLI into a personalized, informative dashboard, significantly enhancing the user experience. Users gain not only an aesthetically pleasing terminal but also immediate access to critical system statistics upon launch, improving efficiency for developers, system administrators, and power users.

The high degree of customizability, managed through simple configuration files, empowers individuals to tailor their terminal's appearance and functionality to their specific preferences and workflows, making the command-line environment more engaging and less daunting.
