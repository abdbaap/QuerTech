Var

Variables

Store and manage data using `let` and `const`. Learn naming conventions.

let count = 10;
const userName = "Alice";
// Naming rules: start with letter, _, or $; no numbers at start; case-sensitive.
Types

Data Types

Understand fundamental types: Numbers, Strings, and Booleans.

let age = 16; // Number
let message = "Hello"; // String
let isActive = true; // Boolean
=/-

Operators

Perform calculations and comparisons with Arithmetic, Comparison, and Logical operators.

5 + 3; // Arithmetic
10 === "10"; // Comparison (strict equality)
(age > 18) && isActive; // Logical
If

Control Flow

Make decisions in your code with `if`, `else if`, and `else` statements.

if (age >= 18) {
  console.log("Adult");
} else {
  console.log("Minor");
}
Loop

Loops

Repeat actions efficiently using `while` and `for` loops.

for (let i = 0; i < 3; i++) {
  console.log("Iteration: " + i);
}
Fn

Functions

Create reusable code blocks. Define with `function` and use `return`.

function greet(name) {
  return "Hello, " + name;
}
console.log(greet("Bob"));
[]

Arrays

Store ordered lists of values. Access elements by their zero-based index.

let colors = ["red", "green", "blue"];
console.log(colors[0]); // Output: "red"
colors.push("yellow"); // Add element
{}

Objects

Organize data with key-value pairs. Access properties using dot notation.

let person = {
  name: "John",
  age: 20
};
console.log(person.name); // Output: "John"
