Seamlessly control your Claude code sessions from Telegram and Discord. Boost your workflow efficiency and code from anywhere.

Get Started Now

Real-Time Interaction, Anywhere

Claude Channels bridges the gap between your coding environment and your favorite messaging apps. Push events into your Claude code sessions, receive outputs, and manage your development workflow directly from your phone.

This feature allows Claude to react to messages from Telegram or Discord, enabling a truly interactive and flexible coding experience. It's event-driven, so events are pushed into your existing session without spawning new ones, ensuring efficiency.

Event-Driven Integration

Receive and process external events directly within your Claude session.

Two-Way Communication

Claude can read events and reply back through the same messaging channel.

Always-On Ready

Process commands in the background for unattended, continuous operation.

Effortless Setup Process

Setting up Claude Channels is straightforward. You'll leverage tools like Telegram's BotFather to create your bot, install the necessary Claude plugins, and then configure your bot with the token obtained.

  • 1

    Create Bot via BotFather

    Interact with `@BotFather` on Telegram to generate your unique bot token.

  • 2

    Install Claude Plugins

    Navigate to your project directory and use the Claude CLI to add the official Telegram plugin.

  • 3

    Configure & Pair

    Configure the plugin with your bot token and follow the prompts to pair your Telegram account.
