Practical Project Demonstration (PDF to Excel Conversion & Cleaning)

This section walks through a real client scenario to demonstrate the process step-by-step.

Client Scenario:

A client required 15 scanned PDF invoices, each containing multiple invoices, to be converted into a single, clean, tabular Excel format.

Key Tools Used:

  • I Love PDF (ilovepdf.com): A free, fast online platform recommended for merging and converting PDFs.
  • Microsoft Excel: Utilized for advanced data cleaning, separation, and formatting.

Step-by-Step Process:

  1. Merge PDFs

    All 15 individual PDF invoices were merged into a single PDF file using the "Merge PDF" tool on ilovepdf.com.

  2. Convert PDF to Excel

    The merged PDF was then converted to Excel using ilovepdf.com, selecting the "One sheet" layout option to consolidate all data into a single Excel worksheet.

  3. Initial Data Review & Demerging

    Upon opening the converted Excel file, it often appears messy with merged cells and unorganized data. The first step involves selecting all cells and using Excel's "Merge & Center" function (under the Home tab) to unmerge any cells, preparing the data for structured cleaning.

  4. Text to Columns for Data Separation

    The main data column (which contained invoice details separated by single pipe | delimiters) was selected. Using the "Text to Columns" feature (Data tab -> Data Tools), I chose "Delimited" and specified "Other" with a | (pipe) character as the delimiter. This efficiently split the combined data into separate, distinct columns (e.g., Item Name, Quantity, Rate, Amount).

  5. Manual Data Correction & Deletion

    Small portions of data that might still be misaligned or in incorrect columns (e.g., data from original Column D appearing in Column G after Text to Columns) were manually cut/pasted or re-typed. Any entirely empty or extraneous columns resulting from the separation were deleted.

  6. Removing Unnecessary Spaces

    To ensure cleanliness, I performed a "Find and Replace" (CTRL+H) operation across the relevant column(s) to replace any multiple spaces with single spaces, or leading/trailing spaces with nothing, as needed.

  7. Standardizing Font and Size

    To ensure a consistent and professional look, all data (CTRL+A) was selected, and the font was standardized to "Calibri" with a size of "12."

  8. Adding Headers

    The original headers (Invoice No, Invoice Date, Item Name, Qty, Rate, Amount) were copied from the original PDF source and pasted into the first row of the Excel sheet. A new row was inserted using ALT+I+R for this purpose.

  9. Converting to an Excel Table

    The entire cleaned dataset, including headers (CTRL+A), was converted into an official Excel Table (CTRL+T). This improves data management, filtering, and analysis capabilities.

  10. Adjusting Row Height

    To further standardize visual presentation, all rows were selected, and their height was auto-adjusted by double-clicking on the boundary between any two row numbers.

  • Master Excel Data Tools: Proficiency in Excel features such as "Text to Columns," "Find and Replace," and "Convert to Table" is essential for quick and accurate data cleaning and formatting.
  • Focus on Client Requirements: Always prioritize delivering data in a "clean tabular format," as this is a primary client need and differentiator in the freelancing market.
  • Seek Practical Experience: Actively engage with practical projects and exercises to solidify understanding and build a portfolio, mimicking real-world client demands.
