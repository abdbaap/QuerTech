Step-by-Step Server Setup Guide

1

Create Server Directory

Establish a dedicated, easily accessible folder on your PC (e.g., C:\Minecraft Server) to house all your server files. This keeps everything organized.

2

Place & Rename Server Jar

Move your downloaded SpigotMC .jar file (e.g., spigot-1.19.2.jar) into your new server directory. Immediately rename this file to server.jar for easier command line usage.

3

Create Start Script

In the server directory, create a new text document. Paste the following command into it:

java -Xms1G -Xmx2G -jar server.jar nogui

Save this file as Start Server.bat. Adjust -Xms1G (initial) and -Xmx2G (maximum) to allocate RAM, matching your PC's available resources (e.g., "1G to 2G").

4

Initial Server Run

Execute the Start Server.bat file. The server will attempt to start, generate essential configuration files (like eula.txt and server.properties), and then close automatically.

5

Accept EULA

Locate and open eula.txt within your server directory using Notepad. Change the line eula=false to eula=true to accept Minecraft's End User License Agreement. Save the file.

6

Configure Server Properties

Open server.properties with Notepad. Important modifications include:

  • online-mode=false: Allows players using non-official launchers (like TLauncher) to join.
  • max-players=70: Adjust the maximum player count (e.g., to "70").
  • motd=EXPOSUREEE Ka Minecraft Server: Customize your server's Message of the Day.
  • Other properties like gamemode, difficulty, and view-distance can also be tailored to your preference.

Enabling External Access for Friends

Connect Friends with Playit.gg (No Port Forwarding!)

  1. Download & Run Playit.gg Agent: Download and execute playit.exe from Playit.gg. This will open a browser window for account setup.
  2. Account Setup: Sign in or create a Playit.gg account.
  3. Add Agent & Tunnel: Click "Add Agent" and then select "Minecraft Java" as the Tunnel Type.
  4. Configure Tunnel: Set Local IPV4 to 127.0.0.1 and Local Port to 25565.
  5. Share Public Address: Create the tunnel and copy the provided public address (e.g., max-handled.craft.ply.gg). Share this address with your friends so they can join your server directly!

Essential Server Management Commands

  • Granting Operator (OP) Status: In the server console (the black command window), type op [player_name] (e.g., op EXPOSUREEE) to give a player administrative privileges.
  • Stopping the Server: Always type stop in the server console to gracefully shut down the server, ensuring all player progress and world data are saved.

Installing Plugins for Enhanced Gameplay

To extend your server's functionality with plugins (requires SpigotMC or PaperMC):

  1. Place Plugin Jar: Download your desired plugin's .jar file and place it directly into the plugins folder within your Minecraft server directory.
  2. Restart Server: After placing the plugin, restart your server by running Start Server.bat. The server will automatically install and activate the plugin.
